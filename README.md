# Puzzle-Game

I created a puzzle game, named Terra Bomber, using the Godot game engine. The game hinges on the idea of destructible puzzles, where levels consist of bombs that players must strategically detonate to clear a path to the goal (a chest). In later levels I was able to build on my foundational mechanics in a modular way that allowed me to implement new ideas such as bomb upgrades and bomb resistant blocks. This project came out much more functional and polished than I was expecting, but I nonetheless think game development is not something I will pursue further. 

As mentioned, Terra Bomber was the result of my learnings about the world of game development.
